◆ registerTransferInDomain()

void ethercat_interface::EcMaster::registerTransferInDomain ( const std::vector< EcTransferNet > &  transfer_nets)

Fill in the EcTransferInfo structures.

Parameters
transfer_netstransfer nets
Precondition
DomainInfo and domain_regs vectors must have been initialized and activated. A call to EcMaster::activate() is required before calling this function, to fill in the domain_regs vector offsets. Specifically with IgH EtherCAT Master, the offset must have been initialized with the ecrt_domain_reg_pdo_entry_list function.
Exceptions
std::runtime_errorif some domain_info or some pdo_entry_reg are not valid
